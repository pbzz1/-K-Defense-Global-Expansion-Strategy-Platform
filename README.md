# 중간팀플 웹사이트 프로젝트

이 저장소는 `backend`, `frontend`, `assets`를 포함한 중간팀플 웹사이트 통합 프로젝트입니다.

## 구성
- `frontend`: React 기반 프론트엔드
- `backend`: Python 기반 백엔드 API/서비스
- `assets`: 시각화 결과, 원본 자료 등 프로젝트 에셋

## 참고 자료
- 프로젝트 기획/전략 참고 문서:
  - `docs/reference/k_방산 글로벌화 전략 (박사님과 싱싱미역줄기들).pdf`

## 주의사항
- 민감정보 및 대용량 불필요 파일은 업로드에서 제외했습니다.
  - `.env`, `node_modules`, `*.zip`
